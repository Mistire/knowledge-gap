# Week 12 — Knowledge Gap Research

Daily paired research, public explainers, and portfolio depth.

Each day: surface one real gap from existing work → sharpen it with a partner → research and write a public explainer for their gap → ship publicly.

## Structure

```
pair_DAY_N/
├── question.md             # Final sharpened question + artifact connection
├── morning_call_summary.md # What was ambiguous, how the question was sharpened
├── explainer.md            # 600–1000 word blog post written for partner's question
├── thread.md               # 4–6 tweet thread
├── evening_call_summary.md # Feedback given and what was revised
├── signoff.md              # Gap closure judgment (closed / partially / not closed)
├── grounding_commit.md     # Pointer to the actual edit made to Week 10/11 portfolio
└── sources.md              # Canonical papers, tools, and patterns used
```
